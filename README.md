# ShoppingCart
Shopping cart
# This Java Program builds a checkout system for a shop which only sells apples and oranges
* • Apples cost 60p and oranges cost 25p
* • Build a checkout system which takes a list of items scanned at the till and outputs the total cost
* - For example: [ Apple, Apple, Orange, Apple ] => £2.05
*
* • This program also caters two new offers in checkout system
* - Buy one, get one free on apples
* - 3 for the price of 2 on oranges

# This Repository also contains the JUnit test classes to test the Java Program builds a checkout system for a shop which only sells apples and oranges
